# rtfToHTML
PHP and JAVASCRIPT LIBRARY TO CONVERT RTF TO HTML


## Runing PHP Project

To run this project, you will need to open the terminal 

Go to the php directory

```bash
  cd php
```

Run index.php 

```bash
  php index.php
```

## Runing NODEJS Project

To run this project, you will need to open the terminal 

Go to the js directory

```bash
  cd js
```

Install Dependency

```bash
  npm install
```

Run index.js 

```bash
   node index.js
```
